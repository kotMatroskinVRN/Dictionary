Run programm

./run.bat

####################################################
Create new Dictionary

create .dct file with

word = description
two words = description

Several words in KEY part separate by " " .

Put this properties file in run directory
Create encrypted dictionary
> ./run.bat -encode your_file.dct
Please do not use ./ or .\ in file name

Result file - encoded_your_file.dct copy to DictionarySource directory
will be copied to DictionarySource and entry of dictionary will be added to EncryptedFiles.txt file
